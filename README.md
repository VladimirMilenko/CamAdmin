CamAdmin
=====================

### Run the example

```
npm install
npm start
open http://localhost:3000
```

### Setup

Install webpack globally.

```
sudo npm i -g webpack
```

### Build
#### Dev

```
webpack --config webpack.config.js
```

#### Prod
```
webpack -p --config webpack.prod.config.js
```

Prod build will be in /docs/ folder
